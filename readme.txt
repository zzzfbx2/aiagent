Financial AI Agent

This project is a Simple Financial AI Agent built using Streamlit. It provides an intuitive user interface for tracking stock prices, analyzing trends with technical indicators, and generating basic buy/sell signals based on moving averages.

Features

Fetch historical stock data using the Yahoo Finance API.

Display raw data for analysis.

Calculate and plot Simple Moving Averages (SMA) for trend identification.

Generate buy/sell signals based on SMA crossovers.

Visualize stock data and signals interactively.

Getting Started

Prerequisites

Ensure you have the following installed:

Python 3.7 or higher

pip (Python package manager)
